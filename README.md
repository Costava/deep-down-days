# Deep Down Days

Endless cave running. JavaScript, HTML, and CSS game.

Performance may be poor in Firefox unless the window resolution is low.

## Controls

- Move: `←` `→`
- Jump: `Z`, `↑`, `Space`
- Hold to Sprint: `Shift`
